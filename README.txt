operations implemented on this project
MOVA		MOVB

SUB		LDI

AND		BRZ
OR		BRN
XOR		JMP
NOT

step to run the project:
- create new project at quartus with top level named "top"
- import every vhd file to project
- set file top.vhd as top level
- compile
- Tools > Simulation Tools > RTL Simulation
